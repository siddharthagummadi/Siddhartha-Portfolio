This is a responsive personal portfolio website built using HTML, CSS, and JavaScript.
It showcases my projects, skills, and contact information with a clean and modern design that adapts to all screen sizes.
Features:
Fully responsive design for desktop, tablet, and mobile
Clean and modern user interface
Sections for About, Skills, Projects, and Contact
Smooth scrolling and interactive elements
Technologies Used:
HTML
CSS
JavaScript
How to Run:
Download or clone the repository
Open the folder in your code editor
Run the “index.html” file in your web browser
Project Structure:
index.html — main file
style.css — styling file
script.js — JavaScript for interactivity
